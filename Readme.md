## вңҰ Vista General

**IPTV Control Center** es un panel administrativo de uso interno construido con FastAPI y Jinja2. EstГЎ diseГұado para operar negocios de IPTV de forma profesional: desde el alta de clientes hasta el seguimiento de renovaciones crГӯticas y el control de ingresos.

> **Uso exclusivamente interno.** No tiene registro pГәblico. El acceso estГЎ controlado por roles y autenticaciГіn JWT vГӯa cookie HttpOnly.

### Capacidades principales

| ГҒrea | DescripciГіn |
|---|---|
| рҹ“Ҡ Dashboard | MГ©tricas en tiempo real: activos, vencidos, ingresos del mes, ventas del dГӯa |
| рҹ‘Ө Clientes | CRUD completo con etiquetas, bГәsqueda y vista de historial |
| рҹ“ә Suscripciones | GestiГіn con cГЎlculo automГЎtico de estado por fecha |
| рҹ“Ұ Planes | CatГЎlogo de planes con activaciГіn/desactivaciГіn |
| рҹ’° Pagos | Registro y confirmaciГіn de pagos multi-moneda |
| рҹ”” Renovaciones | Vista crГӯtica de clientes que vencen hoy, en 3 dГӯas o ya vencidos |
| рҹ‘Ҙ Usuarios | GestiГіn de accesos desde la web (solo superadmin) |
